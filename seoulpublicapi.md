## 서울 지하철 공공 데이터 API 정리

### 서울시 지하철 실시간 도착정보

#### [API Path] http://swopenAPI.seoul.go.kr/api/subway/{KEY}/json/realtimeStationArrival/{START_INDEX}/{END_INDEX}/{STATNMN}

#### [Path Variable] 
- KEY: 인증키
- START_INDEX: 페이징 시작번호
- END_INDEX: 페이징 끝번호
- STATNMN: 지하철역명

#### [RESPONSE]
|출력명|출력설명|
|---|-|
|subwayId|지하철호선ID (1001:1호선, 1002:2호선, 1003:3호선, 1004:4호선, 1005:5호선, 1006:6호선, 1007:7호선, 1008:8호선, 1009:9호선, 1061:중앙선, 1063:경의중앙선, 1065:공항철도, 1067:경춘선, 1075:수인분당선, 1077:신분당선, 1092:우이신설선, 1093:서해선, 1081:경강선, 1032:GTX-A)|
|subwayNm|지하철 호선명|
|updnLine|상하행선 구분 (상행/내선, 하행/외선)|
|trainLineNm|도착지방면 (목적지역 - 다음역 방면)|
|subwayHeading||
|statnFid|이전 지하철역 ID|
|statnTid|다음 지하철역 ID|
|statnId|지하철역 ID|
|statnNm|지하철역명|
|trainCo||
|trnsitCo|환승 노선 수|
|ordkey|도착예정열차순번 (상하행코드(1), 열차순번(1), 이전역~현재역(3), 목적지역, 급행여부(1))|
|subwayList|연계 호선 ID 목록 (예: 1002,1007)|
|statnList|연계 지하철역 ID 목록 (예: 1002000233,1007000000)|
|btrainSttus|열차 종류 (급행, ITX, 일반, 특급)|
|barvlDt|열차 도착 예정 시간 (초)|
|btrainNo|열차 번호|
|bstatnId|종착 지하철역 ID|
|bstatnNm|종착 지하철역명|
|recptnDt|열차 도착 정보 생성 시각|
|arvlMsg2|첫 번째 도착 메시지 (도착, 출발, 진입 등)|
|arvlMsg3|두 번째 도착 메시지 (예: ○○역 도착, 12분 후(△△역))|
|arvlCd|도착 코드 (0:진입, 1:도착, 2:출발, 3:전역출발, 4:전역진입, 5:전역도착, 99:운행중)|
|lstcarAt|막차 여부 (1:막차, 0:아님)|

### 서울시 지하철 실시간 열차 위치정보

#### [API Path] http://swopenAPI.seoul.go.kr/api/subway/(인증키)/xml/realtimePosition/{START_INDEX}/{END_INDEX}/{SUBWATNM}

#### [Path Variable]
- KEY: 인증키
- START_INDEX: 페이징 시작번호
- END_INDEX: 페이징 끝번호
- SUBWATNM: 지하철호선명

#### [RESPONSE]
|출력명|출력설명|
|---|---|
|subwayId|지하철호선ID (1001: 1호선, 1002: 2호선, 1003: 3호선, 1004: 4호선, 1005: 5호선, 1006: 6호선, 1007: 7호선, 1008: 8호선, 1009: 9호선, 1063: 경의중앙선, 1065: 공항철도, 1067: 경춘선, 1075: 수인분당선, 1077: 신분당선, 1092: 우이신설선, 1032: GTX-A)|
|subwayNm|지하철호선명|
|statnId|지하철역ID|
|statnNm|지하철역명|
|trainNo|열차번호|
|lastRecptnDt|최종수신날짜|
|recptnDt|최종수신시간|
|updnLine|상하행선구분 (0: 상행/내선, 1: 하행/외선)|
|statnTid|종착지하철역ID|
|statnTnm|종착지하철역명|
|trainSttus|열차상태구분 (0: 진입, 1: 도착, 2: 출발, 3: 전역출발)|
|directAt|급행여부 (1: 급행, 0: 아님, 7: 특급)|
|lstcarAt|막차여부 (1: 막차, 0: 아님)|

### 서울교통공사_노선별 지하철역 정보

#### [API Path] http://openapi.seoul.go.kr:8088/{KEY}/json/SearchSTNBySubwayLineInfo/{START_INDEX}/{END_INDEX}/{STATION_CD}/{STATION_NM}/{LINE_NUM}

#### [Path Variable]
- KEY: 인증키
- START_INDEX: 페이징 시작번호
- END_INDEX: 페이징 끝번호
- STATION_CD: 지하철역 코드
- STATION_NM: 지하철역명
- LINE_NUM: 호선

#### [RESPONSE]
|출력명|출력설명|
|---|---|
|STATION_CD|전철역코드|
|STATION_NM|전철역명|
|STATION_NM_ENG|전철역명(영문)|
|LINE_NUM|호선|
|FR_CODE|외부코드|
|STATION_NM_CHN|전철역명(중문)|
|STATION_NM_JPN|전철역명(일문)|

#### 예시
```json
{
  "STATION_CD": "0433",
  "STATION_NM": "사당",
  "STATION_NM_ENG": "Sadang",
  "LINE_NUM": "04호선",
  "FR_CODE": "433",
  "STATION_NM_CHN": "舍堂",
  "STATION_NM_JPN": "サダン"
}
```
